## Windows

```text

```

Install:

1. Double-click the installer.
2. If Windows shows "Windows protected your PC", click "More info".
3. Click "Run anyway".
4. Open Tonicue 瞳休 after installation.

Notes:

- Closing the main window does not quit the app. Tonicue keeps running in the system tray.
- Right-click the tray icon to open the main panel, pause reminders, focus-pause for 30 minutes, test a reminder, or quit.

## macOS Apple Silicon

Send this file:

```text

```

Install:

1. Open the `.dmg`.
2. Drag Tonicue 瞳休 into Applications.
3. Open it from Applications.
4. If macOS cannot verify the developer, go to System Settings -> Privacy & Security, then choose "Open Anyway".

Notes:

- This is an unsigned preview build, so security warnings are expected.
- Tonicue stays in the menu bar when the main window is closed.
