# Contributing

Thanks for considering a contribution to Tonicue 瞳休.

This project is an early desktop MVP, so product feedback, UX notes, bug reports, documentation improvements, and small code changes are all useful.

## Development Setup

```bash
npm install
npm run dev
```

Run the current structure check:

```bash
npm run lint
```

## Pull Requests

Before opening a pull request:

- Keep changes focused and easy to review.
- Describe the user-facing behavior you changed.
- Mention how you tested it.
- Avoid unrelated formatting churn.

## Good First Contributions

- Improve README screenshots and usage examples
- Add tests for reminder timing behavior
- Split `src/main/main.js` into smaller modules
- Improve the app icon
- Improve copywriting for reminder messages
- Add accessibility and keyboard polish

## Reporting Bugs

Please include:

- Operating system and version
- Tonicue version
- What you expected
- What happened
- Steps to reproduce
- Screenshots or screen recordings if useful
